# React Template

The React template creates a modern React application using Vite.

## Features

- ⚡️ Lightning fast builds with Vite
- 🎨 CSS Modules support
- 📦 TypeScript ready
- 🧪 Testing with Vitest
- 🎯 ESLint + Prettier

## Project Structure

```
my-app/
├── src/
│   ├── components/
│   ├── App.tsx
│   └── main.tsx
├── public/
├── tests/
├── package.json
└── vite.config.ts
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm test` - Run tests
